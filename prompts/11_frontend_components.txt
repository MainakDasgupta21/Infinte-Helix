Create 20+ React components organized by feature area. Each component uses

and Framer Motion for animations.

--------------------------------------------------------------------------------
Sidebar.jsx — NAVIGATION SIDEBAR (~332 lines)
--------------------------------------------------------------------------------

Fixed-position left sidebar with two states: collapsed (w-16) and expanded
(lg:w-56).

Structure:
  - Logo/brand at top: "IH" monogram (collapsed) / "Infinite Helix" (expanded)
  - Navigation links (vertical list):
    * Dashboard (HiOutlineHome icon) -> /dashboard
    * Tasks (HiOutlineClipboardList) -> /todos
    * Reports (HiOutlineChartBar) -> /reports
    * Cycle Mode (custom flower icon) -> /cycle-mode
    * Calendar (HiOutlineCalendar) -> /calendar
    * Settings (HiOutlineCog) -> /settings
  - Each link: active state with accent-colored left border + background tint
  - Bottom section: user avatar/initials, logout button
  - Active route detection via useLocation()
  - Smooth width transition between collapsed/expanded
  - Mobile: overlay sidebar with backdrop

Styling:
  - bg-helix-surface border-r border-helix-border/50
  - z-40 fixed left-0 top-0 h-screen
  - Icons: w-5 h-5, text-helix-muted (inactive), text-helix-accent (active)
  - Labels: text-sm font-semibold, hidden when collapsed

--------------------------------------------------------------------------------
BreakReminderToast.jsx — BREAK REMINDER OVERLAY
--------------------------------------------------------------------------------

Full-screen or bottom-right toast that appears when a break is due.

Props/Data: uses useBreakReminder() hook

Content:
  - "Time for a break!" heading
  - Supportive message: "You've been working for X minutes. A short break
    will help you recharge."
  - Suggested activities: stretch, hydrate, eye rest
  - "Take a Break" button (starts break timer)
  - "Dismiss" button (snooze for next interval)
  - Timer display when break is active

Styling:
  - Glass-card effect with backdrop-blur
  - Slide-up animation (Framer Motion)
  - Semi-transparent overlay background
  - helix-mint accent color for break theme

--------------------------------------------------------------------------------
StressInterventionToast.jsx — STRESS CALMING OVERLAY
--------------------------------------------------------------------------------

Full-screen calming overlay that appears when stress is detected.

Props/Data: uses useStressDetector() hook

Trigger: appears when stress level exceeds threshold

Content:
  - Calming gradient background (soft blue/purple mesh)
  - "Let's take a moment" heading
  - Guided breathing exercise:
    * Animated circle (breathe animation, scale 1 -> 1.05)
    * Phase text: "Breathe in..." (4s) -> "Hold..." (4s) -> "Breathe out..." (6s)
    * Cycle counter
  - "I feel better" button to dismiss and resolve stress
  - "Continue working" option

Styling:
  - Fixed inset-0 z-50
  - Dark overlay with gradient radial glow
  - Large centered breathing circle with pulse-glow animation
  - Smooth fade-in transition

--------------------------------------------------------------------------------

Circular progress ring showing the wellness score (0-100).

Props: score, streakDays, mood, breaks

Visual:
  - SVG circle (viewBox="0 0 120 120")

Donut chart showing screen time breakdown by app category.

  - breakdown values -> chart data

HydrationTracker.jsx — WATER INTAKE CARD
--------------------------------------------------------------------------------
