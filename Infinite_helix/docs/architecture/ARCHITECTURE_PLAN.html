Core Feature Modules

Work Behavior Tracker

Background Python service using psutil & pynput. Tracks keyboard activity, idle time, screen time, and typing intensity.

AI Emotion Engine

HuggingFace distilroberta for emotion detection, RoBERTa for sentiment. Real-time analysis of journal entries.

Smart Hydration

Behavior-based reminders triggered after long tasks, intense typing, or 2-hour gaps. Not time-based.

Calendar Integration

Google Calendar API v3. Detects meetings 10-15 min ahead and triggers pre-meeting calm nudges.

Cycle Energy Mode

Optional, private cycle phase logging. Adjusts wellness recommendations based on energy patterns.

Notification System

Browser Notification API + desktop toast. Calm, non-intrusive nudges with configurable frequency.

Tech Stack Summary

Frontend

  • React 18 + React Router
  • Tailwind CSS
  • Chart.js (react-chartjs-2)
  • Axios for API calls
  • Browser Notification API

Backend

  • Python 3.10+
  • Flask + Flask-CORS
  • HuggingFace Transformers
  • psutil + pynput
  • Google API Client

Database

  • Firebase Firestore
  • Firebase Authentication
  • firebase-admin (Python)
  • firebase SDK (React)

AI Models

  • j-hartmann/emotion-english-distilroberta-base
  • cardiffnlp/twitter-roberta-base-sentiment
  • PyTorch runtime
